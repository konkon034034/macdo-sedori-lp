# マクドせどりまん プロジェクト固有ルール
**（※本プロジェクトはルート階層の `GEMINI.md` に定義された「完全0コスト運用（API禁止）」「禁止ワードガードレール」を包含し、絶対遵守すること）**
## 1. ビジョンとターゲット
- **目的**: 飲食店（カフェ等）でPCとスマホ1台で行う電脳中古家電せどりビジネスの集客と教育、フロントエンド構築。
- **最終コンバージョン（マネタイズの建て付け）**:
  - LP等を通じて「激アツ商品リスト3選」を無料プレゼント。
  - 最終的に「**残り27個の有料級リストを無料でプレゼントするから、俺の紹介リンク（アフィリエイト等）からコミュニティやサービスに登録してね！**」というオファー（特典・プレゼント）に繋げる。
  - ※注意: **「27個のリストを有料で販売する」という表現は絶対に使用しないこと。これらは紹介経由での無料プレゼントである。**

## 2. デザイン・トーン＆マナー
- ギャル全開、勢い重視。「Wow」なリッチなデザイン。インパクトと親しみやすさを両立させること。
- LPやSNS投稿のテキスト生成においては、「稼げる感」「手軽さ」をアピールする。

## 3. 自動化スクリプト (`auto_sns_poster` との依存関係)
- 当プロジェクトで `auto_sns_poster` をデプロイする際は、必ず `--project macdo` スイッチを使用し、`config/macdo.yaml` に定義されたLPのURLと専用Google DriveのIDを使用すること。
